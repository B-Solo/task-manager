# Veteran Typewriter — font license notes

The scoreboard uses **Veteran Typewriter**, designed by **Magique Fonts** (also published as Typesgal on [DaFont](https://www.dafont.com/veteran-typewriter.font)).

## Files in this directory

| File | Purpose |
| ---- | ------- |
| `veteran_typewriter-webfont.ttf` | Primary file for Qt (`QFontDatabase`) |
| `veteran_typewriter-webfont.woff2`, `.woff`, `.eot`, `.svg` | Web formats from upstream bundle |
| `veteran_typewriter.css` | Upstream `@font-face` declaration |

The webfont bundle was generated by [Font Squirrel](https://www.fontsquirrel.com) (noted in `veteran_typewriter.css`).

## License status

DaFont lists this font as **100% Free** — free for personal and commercial use. Third-party font sites consistently describe it the same way. The upstream VodBox repo (MIT) includes these files but does **not** ship a separate `license.txt` from the font author.

For this private friends-only project, using the vendored TTF is appropriate. If the project is ever shared publicly, re-check [DaFont](https://www.dafont.com/veteran-typewriter.font) or contact the designer for any updated terms.

## Runtime

The Viewer loads `veteran_typewriter-webfont.ttf` at startup and registers it for score numbers on the wax seal. No webfont URLs or CDN access is used at show time.
